# MKM Cubiertas - Landing Page

Landing page profesional para cotización de materiales de construcción ligera (WPC, Deck, Wallcladding). 
Desarrollada con **Node.js**, **Express** y **EJS**.

## 🚀 Características
- Catálogo interactivo de materiales.
- Selección múltiple de productos mediante checkboxes y clics en tarjetas.
- Formulario de contacto funcional con validación.
- Diseño responsivo y moderno (Arquitectónico).

## 🛠️ Tecnologías utilizadas
- **Backend:** Node.js, Express.
- **Frontend:** EJS (Embedded JavaScript templates), CSS3, Vanilla JavaScript.
- **Entorno de desarrollo:** Arch Linux + Neovim.

2. Instalar dependencias:
